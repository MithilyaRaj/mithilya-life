# Mithilya Raj Portfolio

This repository is ready to publish through GitHub Pages at:

`https://mithilyaraj.github.io/mithilya-life/`

## Upload these files

Upload everything in this folder to the root of the `mithilya-life` repository. Keep the folders exactly as they are, including `.github`, `src`, and `public`.

## Publish

1. Open the repository on GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Return to the **Actions** tab and wait for “Deploy portfolio to GitHub Pages” to finish.
5. Open `https://mithilyaraj.github.io/mithilya-life/`.

Every push to the `main` branch redeploys the website automatically.

## Local preview (optional)

```bash
npm install
npm run dev
```
